## Convention

### Code

#### Prettier & Lint

1. 작은 따옴표를 사용한다.
2. 한 줄에 최대 80자로 제한한다.
3. 함수 하나에 최대 10줄로 제한한다.
4. 들여쓰기 깊이를 최대 3개로 제한한다.
5. 매개변수는 최대 2개로 제한한다.

#### Naming

1. 변수명은 카멜 표기법을 따른다.
2. 상수는 대문자로 스크림 스네이크 표기법을 따른다.
3. 배열은 복수형 단어를 사용한다.
